## 5T4R_WEB
모의해킹을 위해 취약한 웹 사이트를 제작하였습니다.

## 목적
1. 개발역량 강화
2. 주요정보통신기반시설 가이드에 대한 이해도 상승
3. 모의해킹 실무 역량 강화
4. 산출물(결과보고서) 작성을 통한 실무 역량 강화
